Project Overview
Singularity is an environmental monitoring network currently in development. The system will consist of multiple autonomous stations distributed across Poland, displaying real-time results on an interactive map.
  • Natural radiation detection via scintillation counter
  • Air quality monitoring with PM2.5 sensor array
  • Electromagnetic pulse detection (custom hardware)
  • UV index measurement and tracking
  • Distributed network with central data aggregation
